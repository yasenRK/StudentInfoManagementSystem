##### 登录模块的设计
* 用MD5来加密用户的密码
* 用一个xml文件来保存登录过用户的密码--当用户勾选了记住登录的时候
* 保存的xml文件中只需要记录登录需要的账号和密码就够了。密码用md5加密的形式保存
